# Two Sum

> Return two indices of number that can be added to get the target number. The same element cannot be used twice.

# Test cases

### Test Case 1

Input :
`nums = [2,7,11,15], target = 9`

Output :
`[0,1]`

### Test Case 2

Input:
`nums = [3,2,4], target = 6`

Output:
`[1,2]`

### Test Case 3

Input:
`nums = [3,3], target = 6`

Output:
`[0,1]`
